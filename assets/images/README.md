# Logo Dosyası

Bu klasöre `logo.png` adında logo dosyanızı ekleyin.

## Önerilen Özellikler:
- Format: PNG (şeffaf arka plan için)
- Boyut: 512x512 piksel (veya daha yüksek çözünürlük)
- Şekil: Kare veya daire (uygulama otomatik olarak uyarlar)

## Dosya Adı:
Logo dosyası `logo.png` olarak adlandırılmalıdır.
